## Project Description
This project analyzes an open source software system and demonstrates
basic Linux scripting skills through five shell scripts.

## Scripts Included

1. script1_system_identity.sh
Displays system information such as kernel version, uptime and user.

2. script2_package_inspector.sh
Checks whether the selected open source package is installed and shows details.

3. script3_disk_auditor.sh
Shows disk usage and permissions for important Linux directories.

4. script4_log_analyzer.sh
Searches a log file for a keyword and counts how many times it appears.

5. script5_manifesto_generator.sh
Creates a short personal open source manifesto based on user input.

## How to Run the Scripts

First give execution permission:

chmod +x scriptname.sh

Then run:

./scriptname.sh
